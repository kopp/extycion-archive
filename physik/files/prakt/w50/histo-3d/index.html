Ich habe das programm hist1.0.cpp geschrieben; damit kann man eine Liste wie 
xy.dat -- diese ist von der Messung der Bewegung mit Laser -- auswerten: Es 
wird ein zweidimensionales Histogramm gezeichnet: Anstatt nur die x- bzw.

liegen. Siehe dazu den Quellcode.

ausgewertet werden -- siehe dazu die .plt Dateien.
